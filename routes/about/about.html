What is gophysics?

gophysics is a set of Physics formulas and constants with the porpuse of simplifying calculations. It is written using the Go programming language, and it is currently a non-stable release.
It covers fields such as Classical Physics, Gravitation, Fluid Dynamics, Thermodynamics, Electromagnetism and Relativity.
The project was developed by me, Gabriele Gatti, an italian Computer Science student with the passion for Physics.
The idea was to create a tool to help other students like me who may struggle in making complex calculations.

Why Go?

I'm really enjoying my experience with this amazing, simple but powerful, programming language. One key aspect of choosing Go was having multi-output functions, which means functions that can return more values at once even of different types. Infact when you use the library each function will return the value and the measurement unit.
Also, as it is rising in popularity, I believe that Go deserves to have more math or Physics related functions.

How can I help?

If you have any suggestion for improving the library you can send an issue in the Github repository, also if this project has been helpful to you consider putting a star, so that more people can participate and use the library.
If you want to add a new feature check how to do that by following the Contribution Guide.
Thank you! :)
